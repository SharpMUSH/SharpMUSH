### Flow 1: Add Banned Name

1. Click **ğŸ”’ Security** â†’ **Banned Names** in sidebar
2. Type name in "Name to Ban" field
3. Click **[+ Add Name]** OR press Enter
4. Name appears in list below
5. âœ… Success notification: "Banned name 'xxx' added"

### Flow 2: Restrict Command to Wizards

1. Click **ğŸ”’ Security** â†’ **Restrictions** in sidebar
2. Ensure **[Command Restrictions]** tab selected
3. Enter command name: `force`
4. Enter restrictions: `wizard, admin`
5. Click **[Add]**
6. Restriction appears in list
7. âœ… Success notification: "Command restriction for 'force' added"

### Flow 3: Block Site from Connecting

1. Click **ğŸ”’ Security** â†’ **Sitelock** in sidebar
2. Enter host pattern: `*.badsite.com`
3. Enter access rule: `!connect`
4. Click **[Add]**
5. Rule appears in configured list
6. âœ… Success notification: "Sitelock rule added"

## Sidebar Behavior

### Auto-Expansion
When you navigate to any security page, the **ğŸ”’ Security** group automatically expands to show:
- Sitelock (with red "Important" badge)
- Banned Names
- Restrictions

### Active Highlighting
The current page is highlighted in cyan (`#00f5b7`) in the sidebar.

### Mobile
On mobile devices (<768px):
- Sidebar collapses to hamburger menu
- Full-width content area
- Tap hamburger to show/hide navigation
